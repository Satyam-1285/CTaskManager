# CTaskManager

A simple, persistent CLI Task Manager written in C.

## Features
* Add tasks with descriptions.
* View all current tasks.
* Mark tasks as completed.
* Persistence: Tasks are saved to `tasks.dat` and reloaded on startup.

## How to Run
1. Compile the code: `gcc main.c -o taskmanager`
2. Run the executable: `./taskmanager`# CTaskManager
